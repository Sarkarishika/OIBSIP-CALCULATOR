# Basic Calculator

A simple and responsive calculator built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations through an interactive user interface and displays both user input and final results on screen.

## GitHub Description

Basic calculator using HTML, CSS, and JavaScript with support for addition, subtraction, multiplication, division, and modulus.

## Features

- Clean and responsive calculator layout
- Interactive button-based interface
- Display screen for input and output
- Supports addition, subtraction, multiplication, division, and modulus
- Includes decimal input handling
- `AC` button to clear all input
- `DEL` button to remove the last character
- Keyboard support for faster input
- Error handling for invalid expressions and division by zero

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How It Works

The calculator accepts numeric input and operators from buttons or the keyboard. JavaScript handles the input logic, validates expressions, and calculates results while respecting operator precedence. The final result is then shown on the calculator display.

## Getting Started

1. Download or clone the repository.
2. Open the `Calculator` folder.
3. Run `index.html` in any web browser.
